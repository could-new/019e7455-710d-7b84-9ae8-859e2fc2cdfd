# Resume Builder

A Flutter-based application that allows users to create and preview their resumes, complete with personal information, work experience, and education details.

## Features
- **Personal Information:** Enter your name, email, phone, and professional summary.
- **Experience:** Add multiple work experience entries.
- **Education:** Add multiple education entries.
- **Preview:** View a clean layout of your resume.
- **Simulated Backend:** The app uses a mock backend service for data persistence across the session.

## Tech Stack
- Framework: Flutter
- Language: Dart
- State Management: `StatefulWidget`

## Setup Instructions
1. Ensure you have Flutter installed (version 3.7.2 or higher).
2. Clone the repository and navigate to the project root.
3. Run `flutter pub get` to install dependencies.
4. Run `flutter run` to start the app on your preferred platform.
